## 💡 نصائح مهمة

### 🔧 تخصيص الألوان
افتح `src/index.css` وغيّر:
```css
--primary-red: #dc143c;    /* الأحمر الأساسي */
--dark-bg: #0a0a0a;        /* الخلفية الداكنة */
```

### 🚗 إضافة سيارات جديدة
افتح `src/components/CarsGallery.jsx` وأضف:
```javascript
{
  id: 5,
  name: 'اسم السيارة',
  year: 2024,
  power: '1000 hp',
  speed: '380 km/h',
  engine: 'V12',
  image: '🏎️',
  color: '#FF0000',
  description: 'الوصف'
}
```

### 📸 استبدال الصور
استخدم Nano Banana لإنشاء صور احترافية:
1. افتح `src/components/CarCard.jsx`
2. بدل الـ emoji بـ صورة URL
3. أضف `<img src="url" />` بدل الـ span

## 🎓 المفاهيم المستخدمة

### Framer Motion
- **motion.div**: عنصر يدعم الحركات
- **initial/animate**: حالات البداية والنهاية
- **whileHover**: تأثير عند التمرير
- **whileInView**: تأثير عند الظهور
- **transition**: مدة وطريقة الحركة

### GSAP
- **gsap.from()**: تأثير من حالة إلى أخرى
- **ScrollTrigger**: حركات عند التمرير
- **gsap.to()**: حركة من الحالة الحالية
- **timelines**: تسلسل الحركات

### CSS المتقدم
- **backdrop-filter**: تأثير الزجاج
- **clip-path**: قص الأشكال
- **@keyframes**: حركات CSS
- **CSS variables**: متغيرات الألوان
