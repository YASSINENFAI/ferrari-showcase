# 📌 تعليمات الرفع على Vercel

## 🚀 خطوات الرفع

### المرحلة 1: التحضير
```bash
# تثبيت Vercel CLI مرة واحدة
npm install -g vercel

# الدخول إلى حسابك
vercel login
```

### المرحلة 2: الانتقال للمشروع
```bash
cd C:\Users\yassine\Desktop\TEST\ferrari-pro
```

### المرحلة 3: الرفع
```bash
vercel
```

---

## ⚡ البديل: رفع سريع جداً

```bash
npm install -g vercel && npm install && vercel
```

---

## 📁 الملفات المتعلقة بالنشر

| الملف | الوصف |
|------|-------|
| `vercel.json` | ⚙️ إعدادات Vercel |
| `.vercelignore` | 🚫 ملفات يتم تجاهلها |
| `deploy.ps1` | 🖥️ سكريبت Windows |
| `deploy.sh` | 🐧 سكريبت Linux/Mac |
