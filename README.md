# 🏎️ Ferrari Showcase Pro

مشروع React احترافي لعرض سيارات فيراري الأسطورية.

## ✨ المميزات

- **Framer Motion**: حركات احترافية وسلسة
- **GSAP**: تأثيرات بصرية متقدمة و Scroll animations
- **Vite**: بناء سريع وتطوير فعال
- **Responsive Design**: يعمل على جميع الأجهزة
- **Modern UI**: تصميم عصري احترافي

## 🚀 البدء السريع

```bash
# تثبيت المكتبات
npm install

# تشغيل التطوير
npm run dev

# بناء الإنتاج
npm run build

# عرض النسخة المبنية
npm run preview
```

## 🎨 التقنيات المستخدمة

- **React 18** - مكتبة واجهة مستخدم
- **Framer Motion** - حركات احترافية
- **GSAP** - تأثيرات متقدمة
- **Vite** - أداة البناء
- **CSS3** - تصميم عصري

## 💡 المكونات الرئيسية

### Navigation
- شريط تنقل ثابت مع تأثيرات Scroll
- أزرار تفاعلية

### Hero Section
- خلفية بجزيئات متحركة
- عنوان رئيسي بتأثيرات
- زر دعوة للعمل

### Cars Gallery
- عرض شبكة للسيارات
- بطاقات تفاعلية مع تأثير 3D
- Scroll animations

### Features Section
- عرض مميزات فيراري
- تأثيرات Hover متقدمة

### CTA Section
- دعوة قوية للحجز
- خلفية ملونة جاذبة

### Footer
- روابط اجتماعية
- معلومات الاتصال
